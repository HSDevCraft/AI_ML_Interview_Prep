## 🎯 Daily Breakdown

### Phase 1: DSA + Python (Days 1-5)

#### Day 1: Arrays, Strings & Python Basics
- Arrays: Two pointers, Sliding window, Prefix sum
- Strings: Manipulation, Pattern matching
- Python: Data types, Memory model, Mutability
- **Practice**: 10 LeetCode problems (Easy/Medium)

#### Day 2: Linked Lists, Stacks & Queues
- LinkedList operations, Cycle detection, Reversal
- Stack/Queue implementations and applications
- Python: Collections module, Generators
- **Practice**: 10 LeetCode problems

#### Day 3: Trees & Binary Search Trees
- Tree traversals (BFS/DFS), BST operations
- Balanced trees concept, LCA problems
- Python: Decorators, Context managers
- **Practice**: 10 LeetCode problems

#### Day 4: Graphs & Advanced Python
- Graph representations, BFS/DFS, Shortest paths
- Topological sort, Union-Find
- Python: Metaclasses, Descriptors, GIL
- **Practice**: 10 LeetCode problems

#### Day 5: Dynamic Programming & Optimization
- DP patterns: 1D, 2D, State machine
- Memoization vs Tabulation
- Python: Async/Await, Threading, Multiprocessing
- **Practice**: 10 LeetCode problems

---

### Phase 2: ML/AI Fundamentals (Days 6-10)

#### Day 6: Math Foundations
- Linear Algebra: Vectors, Matrices, Eigenvalues
- Calculus: Gradients, Chain rule, Jacobian
- Probability: Bayes theorem, Distributions
- Statistics: Hypothesis testing, Confidence intervals

#### Day 7: Supervised Learning - Regression
- Linear Regression, Ridge, Lasso, ElasticNet
- Polynomial regression, Regularization
- Evaluation metrics: MSE, RMSE, MAE, R²
- **Code**: Implement from scratch + sklearn

#### Day 8: Supervised Learning - Classification
- Logistic Regression, Decision Trees, Random Forest
- SVM, KNN, Naive Bayes
- Metrics: Accuracy, Precision, Recall, F1, AUC-ROC
- **Code**: Implement from scratch + sklearn

#### Day 9: Unsupervised Learning & Ensemble
- Clustering: K-Means, DBSCAN, Hierarchical
- Dimensionality Reduction: PCA, t-SNE, UMAP
- Ensemble: Bagging, Boosting (XGBoost, LightGBM)
- **Code**: Implementations

#### Day 10: Feature Engineering & ML Systems
- Feature selection, extraction, transformation
- Handling missing data, imbalanced datasets
- Cross-validation, Hyperparameter tuning
- ML Pipeline design, MLOps basics

---

### Phase 3: Deep Learning (Days 11-15)

#### Day 11: Neural Network Fundamentals
- Perceptron, MLP, Activation functions
- Backpropagation, Gradient descent variants
- Weight initialization, Batch normalization
- **Code**: Build NN from scratch with NumPy

#### Day 12: CNNs & Computer Vision
- Convolution, Pooling, Architectures
- LeNet, AlexNet, VGG, ResNet, EfficientNet
- Object detection: YOLO, Faster R-CNN
- **Code**: PyTorch CNN implementation

#### Day 13: RNNs, LSTMs & Sequence Models
- RNN, LSTM, GRU architectures
- Bidirectional, Attention mechanism basics
- Seq2Seq models, Language modeling
- **Code**: PyTorch LSTM implementation

#### Day 14: Transformers Deep Dive
- Self-attention, Multi-head attention
- Positional encoding, Layer normalization
- BERT, GPT architecture details
- **Code**: Transformer from scratch

#### Day 15: Advanced DL Topics
- GANs: Architecture, Training dynamics
- Autoencoders: VAE, Denoising
- Transfer Learning, Fine-tuning strategies
- Model optimization: Pruning, Quantization, Distillation

---

### Phase 4: GenAI & LLMs (Days 16-18)

#### Day 16: LLM Fundamentals
- GPT, BERT, T5, LLaMA architectures
- Tokenization: BPE, WordPiece, SentencePiece
- Pre-training objectives, Scaling laws
- Inference optimization: KV cache, Speculative decoding

#### Day 17: Prompt Engineering & Fine-tuning
- Prompt engineering techniques
- Few-shot, Zero-shot, Chain-of-thought
- Fine-tuning: Full, LoRA, QLoRA, PEFT
- RLHF, DPO, Constitutional AI

#### Day 18: RAG, Agents & Applications
- Vector databases: Pinecone, Chroma, FAISS
- RAG architecture, Chunking strategies
- LangChain, LlamaIndex frameworks
- AI Agents, Tool use, Function calling
- Evaluation: RAGAS, LLM-as-judge

---

### Phase 5: Interview Prep (Days 19-20)

#### Day 19: Mock Interviews & System Design
- ML System Design patterns
- End-to-end ML project walkthrough
- Behavioral questions preparation
- Resume deep-dive preparation

#### Day 20: Final Revision & Confidence Building
- Quick revision of all concepts
- Common interview pitfalls
- Last-minute tips and strategies
- Mental preparation
